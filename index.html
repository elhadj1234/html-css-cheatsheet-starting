HTML & CSS Comprehensive Cheatsheet

A detailed guide to help you master the basics of web development.

HTML Basics

HTML (HyperText Markup Language) is the standard language for creating webpages. Below is the basic structure of an HTML document:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Page Title</title> </head> <body> <h1>This is a Heading</h1> <p>This is a paragraph.</p> <a href="https://example.com" target="_blank">Visit Example</a> <!-- Comment: This won't appear in the browser --> </body> </html>

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. Below are some basic styles and their purposes:

/* General styles for the page */ body { font-family: 'Arial', sans-serif; /* Sets the default font */ background-color: #f4f4f9; /* Light gray background */ color: #333; /* Dark gray text */ margin: 0; /* Removes default margins */ padding: 0; /* Removes default padding */ } h1 { color: #0078D7; /* Blue heading text */ font-size: 2.5em; /* Larger font size */ } a { color: #0078D7; /* Link color */ text-decoration: none; /* Removes underline */ } a:hover { text-decoration: underline; /* Adds underline on hover */ }

Common HTML Elements

These elements are frequently used in web development. Below are examples:

<ul> <li>First item</li> <li>Second item</li> </ul> <img src="example.jpg" alt="An example image" width="300" height="200"> <table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </table> <form action="/submit" method="POST"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <button type="submit">Submit</button> </form>

CSS Selectors and Advanced Features

CSS selectors are used to target specific HTML elements. Here are some common types:

/* Element selector */ p { color: black; } /* Class selector */ .highlight { background-color: yellow; /* Highlights text */ } /* ID selector */ #main-title { text-align: center; /* Centers the text */ } /* Descendant selector */ div p { font-size: 14px; /* Targets paragraphs inside divs */ } /* Pseudo-class example */ a:hover { color: red; /* Changes link color on hover */ } /* Media query for responsiveness */ @media (max-width: 600px) { body { font-size: 14px; /* Adjusts font size for smaller screens */ } }
